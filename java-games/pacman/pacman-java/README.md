# Java Pac-Man Game 🎮

A simple, classic 2D Pac-Man game built from scratch using Java and Swing. This was created as a fun second-year computer science project to demonstrate basic game loops, 2D array mapping, and collision detection.

## Features ✨
- **Dynamic Game Loop:** Smooth movement updated via a 500ms Timer.
- **Ghost AI:** A pink ghost actively hunts down the player using coordinate-tracking logic.
- **Score System:** Eat food (white dots) to increase your score!
- **Win/Loss Conditions:** If you eat all the dots, you WIN! If the ghost catches you, it's GAME OVER.

## How to Play 🕹️
1. Make sure you have Java installed.
2. Open your terminal or command prompt in the game folder.
3. Compile the game files:
   ```bash
   javac PacManGame.java GamePanel.java
